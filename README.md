# Libros — Marketing y Diseño Gráfico

Repositorio con referencias y notas sobre libros útiles para profesionales y estudiantes.

## Cómo contribuir
- Edita `index.html` desde el editor web de GitHub y añade nuevos bloques de `<article class="book">` con la info del libro.
- Evita subir PDFs con copyright. Enlaza a la editorial, librería o recursos legales.
